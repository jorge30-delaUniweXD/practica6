Frontend

Frontend es la parte del desarrollo web que se encarga de todo lo que el usuario ve e interactúa en un sitio web

Es la interfaz de usuario, el diseño visual, los botones, menús, formularios, etc.

Fundamentos de Frontend

    HTML: Estructura y contenido
    CSS: Diseño y estilos visuales
    JavaScript: Interactividad y dinamismo
